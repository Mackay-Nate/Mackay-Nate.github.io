Creating JavaScript objects and using localStorage

Button 1 will get data from the Meal DB API, https://www.themealdb.com/api/json/v1/1/random.php
Each time the button is pressed, it will create a section and will display the food name, id number, YouTube link, a picture, and the cooking Instructions.
Button 2 will create a meal object, display the object, and store it in localstorage (Button 2 needs to been pushed after Button 1).

Button 3 will create an object from localStorage (from button 2) and display it.
